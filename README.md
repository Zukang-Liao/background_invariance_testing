## Background Challenge Database
This project involves two existing public datasets:
1. the [Background Challenge Database](https://www.dropbox.com/s/0vv2qsc4ywb4z5v/original.tar.gz?dl=0): used to synthesize testing data
2. the IN9(https://www.dropbox.com/s/5bmfiunmlh0hy8n/in9l_resnet50.pt?dl=0) dataset, provided by [IN9_paper](https://arxiv.org/abs/2006.09994): used to train our model repository.
